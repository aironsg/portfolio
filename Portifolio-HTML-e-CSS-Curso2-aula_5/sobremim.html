Sobremim

Olá! Sou Airon Silva, estudante de desenvolvimento mobile com Kotlin, Java e Flutter, e estudante de Ciência da Computação com foco em análise de dados utilizando Python.

Apaixonado por tecnologia desde criança, sempre fui fascinado tanto por jogos offline quanto por tecnologias atuais, como Inteligência Artificial.
Entrei no mundo da programação pela primeira vez em 2020, durante a pandemia da COVID-19. Em meio ao caos mundial, aproveitei para estudar programação através do curso de Análise e Desenvolvimento de Sistemas (ADS). Desde então, venho explorando diversas áreas da tecnologia, como: Backend com Spring Boot, análise de dados com Python e desenvolvimento de aplicativos com as linguagens Kotlin, Java e Dart, utilizando o framework Flutter.

Mas nem só de código vive alguém apaixonado por tecnologia. Sou um homem religioso (Adventista do Sétimo Dia), gosto de jogar games online, assistir filmes e séries e, claro, brincar e ensinar meus filhos sobre tecnologia.

Atualmente, ainda busco minha primeira oportunidade como desenvolvedor de software e estou me especializando na área de dados.

Certa vez, ouvi uma frase que achei intrigante: "Antes de amaldiçoar a escuridão, acenda uma vela." — Confúcio.
E é isso que estou fazendo: acendendo uma vela para iluminar o meu caminho e o caminho daqueles que estiverem ao meu lado.
